圖像辨識機器人

展示影片 github連結(README)

麵包探險家是一個圖像辨識的Line Bot，主要功能是讓使用者傳入麵包圖片後，藉由圖像辨識讓使用者了解該麵包的介紹以及歷史，並推薦給使用者當前位置附近的最近麵包店。
整體專案中，串接了Azure的圖像辨識、Google map API、Open AI API，在過程中，團隊遇到了許多問題，例如：Azure的圖像辨識雖然可以讀取網址，卻無法讀取imgur的圖庫，並在團隊積極溝通下，我們決定放棄使用imgur，轉而將照片直接儲存在本機上，另外則是因為我們雖然串接了chatGPT來進行麵包的說明，但由於chatGPT回答內容並不是完全準確，因此我們設置了fakeGPT，主要是將正確率高的答案進行儲存，在下一次使用者有相同麵包名稱時，直接返回該內容，也能夠更快速的回覆使用者。 在整體專案中，我主要負責Line Bot的串接和整合，將組員提供的代碼進行串接以及Google map API的製作，團隊每個人的工作佔比皆為1/3。

MoodWave網站專案

展示影片   網站網址   Github連結(README)

這是一個讓使用者撰寫日記，並透過 chatGPT 提供即時回應的網站，同時使用者也具有將日記將日記進行下載、分享等功能。
在前端方面，我選擇使用 React 和 Tailwind CSS 來打造界面，包括設計界面、首頁、日記撰寫頁面、查看日記等， 後端則是使用 Express框架構建後端 API來負責處理用戶的不同請求，例如保存和檢索日記數據，另外我使用 MongoDB Atlas 作為後端資料庫來儲存用戶的日記和相關資料，並整合 GPT3.5 模型作為 chatGPT，來增加與使用者的互動性，並最終將網站其部署到 Vercel。
